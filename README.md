# Student Management System

## About the Project

This is a simple Student Management System developed using Python. I created this project to practice basic Python concepts such as functions, dictionaries, loops, conditional statements, and user input handling.

The application runs in the command line and allows users to manage student records easily.

## Features

* Add a new student
* Search for a student by roll number
* Update student marks
* Delete student records
* View all students
* Simple menu-driven interface

## Technologies Used

* Python 3

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Run the Python file:

```bash
python student_management_system.py
```
